ParkSmart

ParkSmart is a Java Swing smart parking management project focused on DSA implementation, role-based access, route finding, slot handling, vehicle tracking, reservations, and analytics.

Java Swing DSA Project Shortest Path Parking Analytics

Overview

The project models a smart parking environment using modular Java packages for GUI, services, models, and DSA logic. It supports route visualization, vehicle and slot history, reservation workflows, and live dashboard metrics.

Key Features

  • Admin and staff login flow
  • Dashboard and analytics panels
  • Slot, vehicle, and reservation management
  • Shortest path route visualizer
  • History tracking and filtering

DSA Concepts

  • Graphs for parking map modeling
  • Shortest path algorithms for route finding
  • Sorting and filtering for records
  • Greedy and optimization ideas in analytics

Run Instructions

javac --release 8 -d out -sourcepath src src/app/Main.java
java -cp out app.Main
